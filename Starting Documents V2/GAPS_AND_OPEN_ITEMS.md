# Gaps & Open Items — Math Meltdown (V2)

This repo shares its textbooks, image library, and font files with
`Starting Documents/` (the source "Math Fun Facts" brand) — only the brand-voice and
prompt-template layer changed. Most gaps below are therefore identical to the source
repo's, plus one new item specific to this V2 build. Nothing here was silently filled
in — this is the punch list.

## Needs a decision from you

1. **Brand color palette isn't set.** `01_Brand_Voice/Math_Content_Brand_Voice_Guide.md`
   Section 8 has a suggested direction (hot pink / acid yellow / electric blue on black)
   so image prompts have something concrete to reference, but it's not confirmed —
   same open status as the source brand's palette.
2. **Font provenance is unverified — new item, specific to the current state of the
   shared Fonts folder.** `02_Visual_Assets/Fonts/` now contains two files
   (`Paraaminobenzoic.otf`, `Styrofoam Feelings.otf`) with no accompanying license file.
   Their license terms can't be confirmed from what's in the repo. Don't treat them as
   cleared for commercial use until someone confirms their source. See
   `05_Compliance/Compliance_Notes.md` §3.
3. **Textbook license vs. monetization.** Same as the source repo — all five reference
   textbooks (`03_Source_Material/Textbooks/`) are CC BY-NC-SA 4.0. Fine for
   fact-checking, not safe to quote verbatim or reproduce diagrams from if this
   account's output is monetized in a way that counts as commercial use.
4. **Every visual asset is unreviewed.** `05_Compliance/License_Attribution_Tracker.csv`
   has 100 rows, all `Not Reviewed` — same shared list as the source repo.

## Real content gaps found (shared with the source repo — same underlying library)

5. **Half the content pillars have little or no matching visual asset library.** The
   100-item asset library maps cleanly onto only 2–3 of the 6 content pillars:
   - Number Chaos — well covered (Number Patterns category).
   - Shape Chaos — well covered (3D Solids, Classical Theorems, Fractals).
   - Nature's Cheat Codes — partially covered (Fractals categories overlap loosely;
     nothing depicting phyllotaxis, shells, coastlines directly).
   - **Dead Mathematician Gossip — no matching assets at all.** No portraits, period
     diagrams, or historical illustrations in the library.
   - **Brain Melters — no matching assets at all.** No Monty Hall, Zeno, infinite-set,
     or similar diagrams.
   - **Math Is Everywhere, Actually — no matching assets at all.** No music/art/
     architecture/daily-life diagrams.

   This gap matters more for this brand, not less — a high-energy, meme-heavy account
   leans even harder on strong visuals than a quiet one does. Worth prioritizing a
   second sourcing pass for these three pillars.

6. **No source material for the Dead Mathematician Gossip pillar at all.** All five
   textbooks are modern OpenStax instructional texts with no historical/biographical
   content. Suggest adding a source before publishing this pillar — e.g. the MacTutor
   History of Mathematics archive (University of St Andrews) — the drama in this
   pillar's voice needs to come from real history, not embellishment (see Section 9 of
   the voice guide).

7. **Thin source material for Nature's Cheat Codes.** No dedicated popular-science
   source for nature claims (phyllotaxis, shell spirals, honeycomb geometry). Calculus
   Volume 1's optimization chapters can support a "nature's cheating" framing, but
   nothing here verifies the specific natural-world claims themselves.

8. **One visual asset failed to download.** Same known gap as the source repo —
   `All Platonic solids & Some Archimedean solids.jpg` 404'd on Commons. See the source
   repo's gaps doc for the two unverified replacement leads that were found.

9. **No alt-text / accessibility guidance.** Same gap as the source repo — worth adding
   regardless of brand voice.

10. **No posting cadence / content calendar.** Same gap as the source repo.

## About this file's structure
The source repo's `GAPS_AND_OPEN_ITEMS.md` currently lists only content gaps, without a
"needs a decision" section. That section is kept here because the underlying decisions
(palette, compliance, font provenance, monetization) are still real open items for this
brand too — arguably more visible ones, given the font-provenance issue found while
building this V2 copy.
