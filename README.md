# AI Reference Proof of Concept Repository

This repository contains sample documents, images, audio, video, spreadsheets, code, and normalized AI-readable derivatives for testing AI-assisted retrieval, summarization, extraction, and reasoning.

## Goals
- Test how AI tools reference different media types.
- Compare raw files vs normalized text.
- Evaluate retrieval accuracy.
- Demonstrate metadata-driven search.
- Prepare for future workplace implementation.
